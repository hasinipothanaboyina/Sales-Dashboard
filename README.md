# Sales & Revenue Analysis Dashboard

An interactive and responsive dashboard for analyzing enterprise sales and revenue data using CSV files.

---

## Features

- CSV File Upload
- Dynamic KPI Cards
- Revenue Trend Analysis
- Industry Performance Analysis
- Interactive Charts
- Year Filter
- Responsive Dashboard UI
- Real-Time Data Processing

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js
- Papa Parse

---

## Dashboard Modules

### KPI Cards
- Total Income
- Total Profit
- Total Records
- Dataset Overview

### Charts
- Revenue Trend Line Chart
- Industry Performance Bar Chart

### Filters
- Year-wise Filter

### Step 1
Download or clone the repository

```bash
git clone https://github.com/your-username/Sales-Revenue-Analysis-Dashboard.git
```

### Step 2
Open the project folder

### Step 3
Run index.html in browser

### Step 4
Upload the CSV dataset
